# College Online Store Website 
College Online Store Website
About the Project

This project is a front-end implementation of an online store website developed during my college studies.
The website demonstrates the structure and layout of an e-commerce platform, including product catalog pages and navigation.

The goal of the project was to practice web layout, page structuring, and user interface organization.

Features

Product catalog page

Individual product pages

Navigation menu

Structured layout for online store

Basic UI/UX principles applied

Responsive page structure

Technologies Used

HTML5

CSS3

Basic responsive layout techniques

Semantic markup

Project Structure
index.html        – Main page  
katalog.html      – Product catalog  
katalog.css       – Catalog styling  
site.css          – Main styles
1-15.html         - product card

Learning Outcomes

Improved understanding of HTML page structure

Practiced CSS styling and layout organization

Learned how to structure multi-page websites

Applied basic UI/UX design principles
